david.sferruzza.fr
==================

This is my personnal website's source code.
It is made with [Hakyll](http://jaspervdj.be/hakyll/).

## How to install

- install haskell platform
- `cabal update`
- `cabal install cabal-install`
- clone this repo
- `cabal sandbox init`
- `cabal install --dependencies-only`
- `cabal build`

To build: `cabal run build`
